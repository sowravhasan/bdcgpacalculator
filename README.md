## 📱 **How to Use**

### **🎯 For Exact CGPA Values (Recommended)**
1. **Navigate** to Overall CGPA Calculator section
2. **Select "Exact CGPA"** mode (default)
3. **Enter semester name** (e.g., "1st Semester", "Spring 2024")
4. **Input your exact CGPA** (e.g., 3.47, 2.59, 3.85)
5. **Click "Add Semester"** - your overall CGPA updates instantly!

### **📊 For Letter Grades**
1. **Choose university** from the dropdown menu
2. **Select "Letter Grades"** mode 
3. **Add subjects** with credit hours and grades
4. **Watch real-time** GPA calculation
5. **Export results** as CSV for records

### **🎯 Target Planning**
1. **Enter current subjects** and grades
2. **Set target CGPA** in the planner
3. **Input remaining credits**
4. **Get exact grades needed** for your goal
